◆ Calibrate() [1/2]

void control::Pose::Calibrate ( int16_t  _num)

@description: naive calibration for all offsets read from IMU and compute the average for given times

Parameters
num, IMU reads to compute the average offset
Note
: it takes _num * 10ms to calibrate

Explanation: For calibration, we want the mean of noise distribution (bias) to be zeroed when IMU is placed on a flat plane. Note that acc_z_off should have a constant gravity offset on a flat plane, which also has a noise in measuring. Since the value of the gravity doesn't matter (only ratio matters), we can assign the noise bias to the gravity noise bias. So acc_z_off is always zero.
